About Project


This application provides basic service for face recognition, accordingly with database collected.
As an input it takes an image and basically checks if there is a face on it. In case it is, the system compares it with faces from it's database and gives a name of the person found or throws a message, that this person is not familiar for it.
One of the most important advantages of this system before analogues is it's ability to learn permanently, i.e. if the person is not yet familiar, the system asks user about the actual name of the face of photo and saves this imformation into database, automatically adding this face into algorithm of face recognition in the future.
